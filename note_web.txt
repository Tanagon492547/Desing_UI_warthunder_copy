css---------------------------------------------------------------------------------------------------------------------------------------

Position Relative คือ Attribute ในการจัดตำแหน่งของ CSS โดยมีคุณสมบัติคือ จะทำการจัดตำแหน่งโดยอิงจาก DIV ที่ห่อหุ้มตัวมันอยู่ ภาพรวมของ Position Relative. 1. ใช้สำหรับการจัดตำแหน่งของ DIV ที่ต้องการ 2. ใช้รวมกับคำสั่ง left, right, top, bottom สำหรับการเคลื่อนตำแหน่งไปในทิศทางต่าง ๆ

position: absolute; เเสดงผลอิสระที่อยู่ภาย  element 

z-index: -1; /* ทำให้ภาพอยู่ด้านหลัง */

background-size: cover; /* ตรงตัวเลยลองทำดูเดี๋ยวก็รู้ */
background-position: center;

 filter: brightness(45%) /* กำหนดให้พื้นหลังดำ ข้อควรระวังใน สมาชิกของมันก็จะดำด้วย*/

 justify-content: space-between; /* เเยกวัตถุออกจากกันอย่างอิสระ */

* a{ /* การลบเส้นใต้ของลิงค์ */
    text-decoration: none;
}

* li{ /* การลบจุด*/
    list-style: none;
}

font-weight: bold; /* ทำให้ตัวอักษรหนา */ 

opacity: .6; /* คำสั่งสำหรับการกำหนดค่าความทึบแสงของ Object หรือ วัตถุต่าง ๆ ที่ปรากฏบนหน้าเว็บไซต์ เช่น รูปภาพ ตัวอักษร หรือ DIV ตามที่เราต้องการ */
